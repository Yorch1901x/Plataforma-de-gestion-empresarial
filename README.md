# Portal Project

Proyecto completo en Node.js con HTML, CSS y JavaScript.

## 📁 Estructura del Proyecto

```
PORTAL/
├── public/              # Archivos públicos (HTML)
│   └── index.html      # Página principal
├── src/                # Código fuente
│   └── server.js       # Servidor Express
├── assets/             # Recursos estáticos
│   ├── css/
│   │   └── styles.css  # Estilos principales
│   └── js/
│       └── main.js     # JavaScript del cliente
├── package.json        # Configuración del proyecto
├── .gitignore         # Archivos ignorados por Git
└── README.md          # Este archivo
```

## 🚀 Instalación

1. **Instalar dependencias**
   ```bash
   npm install
   ```

2. **Iniciar el servidor**
   ```bash
   npm start
   ```
   
   O para desarrollo:
   ```bash
   npm run dev
   ```

3. **Acceder a la aplicación**
   ```
   http://localhost:3000
   ```

- ✅ Estilos CSS modernos con animaciones

- **Node.js** - Runtime de JavaScript
- **Express** - Framework web
- **HTML5** - Estructura
- **CSS3** - Estilos y animaciones
- **JavaScript (Vanilla)** - Interactividad

## 📝 Próximos Pasos

1. Instalar las dependencias: `npm install`
2. Personalizar el contenido en `public/index.html`
3. Añadir más páginas en la carpeta `public/`
4. Crear nuevas rutas en `src/server.js`
5. Expandir estilos en `assets/css/styles.css`
6. Añadir más funcionalidad en `assets/js/main.js`

## 📞 API Endpoints

- `GET /` - Página principal
- `GET /api/health` - Estado del servidor
- `POST /api/contact` - Enviar formulario de contacto
